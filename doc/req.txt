I. Ширина пучка

1. Найти максимум по Y
2. Построить график по Х
3. Найти полуширину (горизонтальный пунктир)

X^
 |     ^
 |   /   \
 |  /-----\
 |_/       \_
 |_________________>
                   n
4. Такой же график по Y, т.е. найти максимум по Х и построить график по Y, посчиать полуширину.
5. В UI добавить две кнопки для вычисления ширины (по Х и по Y).
6. Кроме самоы ширины нужно еще строить график.
7. Нужна возможность строить ширину пучка не по полувысоте, а по 0.75 высоты, например. Т.е. нужно возможность вводить этот параметр.
8. Так же не всегда нужно строить ширину по максимуальному значению. Нужно по клику мышью на график строить ширину пучка по выбранному квадрату.



II. Перевод координат в углы

Нужно заменить координаты на углы.
1. Координаты по Y (y_1, y_2, ..., y_n) заменяются на (y_i - y_0)/L. Пусть это будет угол α.
2. Аналогично координаты по Х. Пусть это будет угол β.
3. х_0, y_0 и L задаются пользователем.
4. Построить хитмапу по новым координатам: (β,α). Или наоборот по (α, β). Сейчас не важно.
5. Концептуально х_0, y_0 и L изображены схематически ниже:

 |                    |(x_0, y_i)
 |                    |
*|------------------->| (x_0, y_0)
 |         L          |
 |                    |

На изображении:
i.   звездочка -- это лазер
ii.  dертикальная линия -- граница колпака,
iii. L -- расстояние от лазера до Галея, который изображен вертикальной линией напротив
     (вся картинка это вид сбоку на установку).
iv.  L вычисляется как сумма расстояния от колпака до Галея плюс расстояние от стенки колпака
     до лазера внутри колпака. Инфы по внутреннему устройтвую Галея нет, т.е. на сколько глубоко
     спрятан приемник внутрь корпуска Галея.
